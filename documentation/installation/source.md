## Requirements

To work with the source code of this Flutter project, you will need the following:

- **Flutter SDK**: Ensure you have Flutter installed on your system. Flutter is a UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase. Download and install Flutter from the official website: [Flutter Installation Guide](https://docs.flutter.dev/get-started/install).
- **GitHub Account**: A GitHub account is required to fork or clone the repository. If you don't have one, sign up at [GitHub](https://github.com).
- **Development Environment**: Visual Studio Code (VSCode) with the Flutter extension is recommended. Install VSCode from [Visual Studio Code](https://code.visualstudio.com/) and add the Flutter extension from the marketplace.
- **Additional Tools**: Depending on your target platform (Android, iOS, etc.), you may need Android Studio or Xcode. Refer to the Flutter documentation for platform-specific setup.

For help getting started with Flutter development, view the [online documentation](https://docs.flutter.dev/), which offers tutorials, samples, guidance on mobile development, and a full API reference.

## Steps to Use the Source Code

1. **Get the source code**:
   - Stable: Download the source folder from **Releases** section of this repository (usually found on the right-hand sidebar of the main page)
   - Preview: Alternatively, click "Fork" on GitHub to create your own copy, or clone the repository directly using Git: `git clone https://github.com/papaang/Project_Impactive_Sleep_Tracker_App.git` to get the lastest version of the code.
   - For more details on forking and cloning, see [GitHub's Forking Guide](https://docs.github.com/en/get-started/quickstart/fork-a-repo).

2. **Open the Project in VSCode**:
   - Launch Visual Studio Code.
   - Open the cloned or forked project folder.
   - Ensure the Flutter extension is installed and enabled. If prompted, allow VSCode to detect the Flutter project.
   - Run `flutter pub get` in the terminal to install dependencies.
   - Navigate to the `lib/main.dart` file
   - To run the app, use `flutter run` or press "Start Debugging" in VSCode.

For further assistance, refer to the [Flutter Development Documentation](https://docs.flutter.dev/).
